# 🪑 FurniVibe Frontend - Modern Furniture Storefront

This is the frontend application for **FurniVibe**, a modern e-commerce platform. It provides a sleek, responsive user interface for customers to browse furniture catalogs, manage their shopping carts, and process orders through the FurniVibe Django API.

## ✨ Features

-   **Product Catalog:** High-performance browsing with filtering by category and brand.
-   **Dynamic Cart:** Real-time shopping cart management and persistent storage.
-   **User Authentication:** Integrated with `SimpleJWT` for secure login, registration, and profile management.
-   **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
-   **Order Tracking:** User dashboard to view order history and shipment status.

## 🛠 Tech Stack

-   **Library:** React.js / Vite
-   **State Management:** Redux Toolkit or Context API
-   **API Client:** Axios (configured for JWT interceptors)
-   **Styling:** Tailwind CSS / Styled Components
-   **Package Manager:** npm

## 📋 Prerequisites

Before you start, ensure you have the following installed:
-   Node.js (v16.0+)
-   npm (v7.0+)
-   A running instance of the FurniVibe Backend

## 📦 Installation

1.  **Navigate to the frontend directory**
    ```sh
    cd F_furniVibe
    ```
2.  **Install dependencies**
    ```sh
    npm install
    ```
3.  **Environment Setup**
    Create a `.env` file in the root directory:
    ```env
    VITE_API_BASE_URL=http://127.0.0.1:8000/api/
    ```

## 💻 Usage

### Development Server
```sh
npm run dev
```

### Production Build
```sh
npm run build
```
