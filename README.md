# FurniVibe

FurniVibe is a modern, high-performance e-commerce platform dedicated to providing a seamless furniture shopping experience. Built with a focus on code quality and scalability, it leverages the latest JavaScript ecosystem tools to deliver a responsive and intuitive user interface.

## 🚀 Features

- **Curated Catalog:** Browse through a wide variety of furniture styles and categories.
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Modern Tech Stack:** Utilizes React, TypeScript, and Babel for robust frontend development.
- **Advanced Search & Filtering:** Quickly find pieces that fit your "vibe" using intelligent pattern matching.
- **Secure Transactions:** Built-in support for secure data handling and UUID generation for order tracking.

## 🛠 Tech Stack

- **Frontend:** [React](https://reactjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Compiler:** [Babel](https://babeljs.io/) (with support for experimental syntax like decorators and class properties)
- **Utilities:** 
    - `node-uuid` for unique identifier generation.
    - `is-glob` / `is-extglob` for advanced file and pattern matching.
    - `word-wrap` for clean text formatting in UI components.

## 📦 Installation

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**
   ```sh
   git clone https://github.com/your-username/FurniVibe.git
   ```
2. **Install NPM packages**
   ```sh
   npm install
   ```

## 💻 Usage

To start the development server:
```sh
npm start
```

To build the project for production:
```sh
npm run build
```

To run tests (using `tap` or your preferred test runner):
```sh
npm test
```

## 📂 Project Structure

- `/node_modules`: Project dependencies.
- `/src`: Source code including components, hooks, and styles.
- `/test`: Test suites for ensuring code reliability.
- `package.json`: Project metadata and script configurations.
